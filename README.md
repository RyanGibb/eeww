# HMAC-based Extract-and-Expand Key Derivation Function (HKDF)

An implementation of [HKDF](https://tools.ietf.org/html/rfc5869) using
[nocrypto](https://github.com/mirleft/ocaml-nocrypto), including test cases from
the RFC.
